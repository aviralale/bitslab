# BitsLab - LLM Context and Information
# Last Updated: 2026-02-04

## Project Overview
BitsLab is a free online utility toolkit designed for creators, developers, and everyday users, with special focus on the Nepal market.

Platform: https://lab.ctrlbits.com
Owner: Ctrl Bits
Repository: https://github.com/aviralale/bitslab

## Project Description
BitsLab provides a collection of productivity and conversion tools accessible directly from the browser. Tools are built with React, TypeScript, and Vite for optimal performance.

## Available Tools

### 1. Nepali Font Converter
- Location: /unicode-preeti-converter
- Purpose: Convert between English, Unicode, and Preeti fonts
- Target Users: Students, designers, publishers
- Features: Bidirectional conversion, instant processing, support for Nepali script

### 2. Text Formatter
- Location: /text-formatter
- Purpose: Clean, format, and manipulate text
- Features: Case conversion, text cleanup, code formatting (camelCase, snake_case, kebab-case), line manipulation
- Metrics: Character count, word count, line count

### 3. QR & Barcode Generator
- Location: /qr-barcode-generator
- Purpose: Create customizable QR codes and barcodes
- Features: Multiple dot styles, logo overlay, color customization, error correction levels, barcode format support
- Use Cases: Product labeling, URL sharing, event ticketing, document tracking

### 4. Invoice Generator
- Location: /invoice-generator
- Purpose: Create professional invoices from the browser
- Target Users: Freelancers, small business owners, consultants
- Features: Professional templates, instant download, no sign-up required
